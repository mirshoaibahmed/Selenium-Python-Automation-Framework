# Selenium Python Automation Framework

An enterprise-grade automation framework built using **Python**, **Selenium**, and **Pytest**. 

## 🚀 Features
- **Page Object Model (POM)**: Enhanced maintainability and reduced code duplication.
- **Data-Driven Testing**: Externalized test data using JSON.
- **Automated Reporting**: Generates self-contained HTML reports with embedded screenshots on failure.
- **Cross-Browser Ready**: Configurable driver setup in `conftest.py`.

## 🛠️ Tech Stack
- **Language**: Python 3.12
- **Framework**: Pytest
- **Tool**: Selenium WebDriver
- **Reporting**: pytest-html

## 📋 How to Run
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Execute tests: `pytest`
